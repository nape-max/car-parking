Задание:
    Клиент:
        1) Созданиие,
        2) Редактирование,
        3) Удаление.
    Автомобиль:
        1) Создание,
        2) Редактирование,
        3) Удаление.

    Дополнительно:
        1) Метод для учёта сколько и какие "Автомобиль" находится на стоянке,
        2) Документацию по api в программе Postman (https://www.postman.com/, файл экспорта коллекции v2.1 добавить в репозирий),
        3) Защита от XSS атак,
        4) Защита от SQL-инъекций,
        5) Соответствие MVC.

    Основные сущности:
        Клиент:
            1) ФИО (мин 3 символа)
            2) Пол (обязательный)
            3) Телефон (обязательный, уникальный)
            4) Адрес 
            5) Автомобил(ь/и) (обязательный, мин 1 шт)
        Автомобиль:
            1) Марка (обязательный)
            2) Модель (обязательный)
            3) Цвет кузова (обязательный)
            4) Гос Номер РФ (обязательный, уникальный)
            5) Флаг статуса автомобиля, означающий, что авто находится или отсутствует на стоянке (обязательный)

    Требования:
        1) PHP 7.4 и выше,
        2) MySQL 8 и выше / MariaDB 10.3 и выше,
        3) Не использовать ORM. Можно использовать QueryBuilder.

    Сдача:
        Hепозиторий на Github, в котором должен быть код и документация методов (достаточно просто запрос + 1-2 примера).
